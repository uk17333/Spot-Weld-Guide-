Power Apps Access Guide

🚀 Getting Started

To access the Spot Weld Concern system, you need Power Apps permissions. Follow these steps:

1 Request Access

Access Request Process:

  1. Go to: Power Apps Access Request
  2. Scroll down to the bottom of the page
  3. Submit the form to gain access to Power Apps
  4. Complete the form that appears
2 Complete the Access Request Form

Fill in the Form Fields:

  • UsageType: Select "Use only" (for logging concerns)
  • Approver: Enter your Supervisor/Senior's name or email address
  • RequestersComments: Enter "Access required for TPQC quality management system"
  • TermsOfUse: Check "Yes" to accept terms
  • Attachments: None required (leave blank)
  • Apply label: Leave as "None"
  • Click "Save" to submit your request
3 Approval Process

What Happens Next:

  • Request goes to your Supervisor/Senior for approval
  • If rejected, escalate to Manager level
  • Will continue through management chain until approved
  • Process typically takes 1-3 business days
  • You'll receive email notification when approved
4 Access the Database

Once Approved:

  1. Navigate to: Spot Weld Database
  2. Click "Spot Weld Count and Position Database" button
  3. Start logging concerns using the "+ New" button

⚠️ Important: Access Required First

You must complete this access request before you can use the spot weld concern system. Without Power Apps access, you won't be able to view or create concerns in the database.

How to Log a Spot Weld Concern - Step by Step

⚠️ Prerequisites

Before following these steps, ensure you have completed the Power Apps access request in the first tab. You cannot log concerns without proper access.

1 Access the Database

Navigate to the System:

  1. Open your web browser (Chrome recommended)
  2. Go to: Spot Weld Database
  3. Click: "Spot Weld Count and Position Database" button
2 Create New Concern

Start New Entry:

  1. Click the "+ New" button (top left of the list)
  2. New form opens on the right side
3 Fill in Required Details

Complete the Form:

  • Fill in all required fields (marked with *)
  • Use dropdown menus where available
  • Enter responsible person's email address
  • Add clear description of the concern

👥 Need Help Finding Responsible Person?

Check the Team Structure to identify the correct person for your area/line.

4 Add Photo

Attach Photo:

  1. Scroll to "Attachments" section
  2. Upload photo of the concern
5 Submit the Concern

Save and Start the Process:

  1. Review all fields are completed correctly
  2. Click the checkmark (✓) button at the top right
  3. Form closes and returns to list view
  4. Your concern appears with Status: "Awaiting Countermeasure"

⚡ What Happens Next (Automatic)

Within 5 Minutes:

  • Status Update: System changes to "Awaiting Countermeasure"
  • Report Generation: HTML report created with details and image
  • Approval Request: Sent to responsible person via Teams Approvals

📝 For Responsible Persons: How to Respond to Approval Requests

When you receive an approval email:

  1. Open the email - Contains all concern details
  2. Click the report link - View full printable report with images
  3. Investigate the concern - Check the actual location/issue
  4. Implement countermeasure - Fix the problem first
  5. Add detailed comments in the approval response explaining:
    • What action was taken
    • When it was completed
    • Any preventive measures implemented
    • Verification that issue is resolved
  6. Click "APPROVE" - Only after countermeasure is complete
  7. Click "REJECT" - If concern needs clarification or is invalid

💡 Good Comment Examples:

Approve: "Extra weld removed using grinder on 10/08/25. Robot program updated to prevent recurrence. Area inspected and approved by supervisor. QA verification completed."

Reject: "Unable to locate described issue. Inspected area thoroughly - no extra welds found. Request clearer photo or more specific location details."

✅ For TPQC Team: Final Confirmation Process

When you receive a confirmation request:

  1. Review original concern - Check initial problem details
  2. Evaluate countermeasure - Assess if action taken is adequate
  3. Verify implementation - Confirm work was actually completed
  4. Add TPQC comments explaining your decision
  5. Click "APPROVE" - If countermeasure meets TPQC standards
  6. Click "REJECT" - If countermeasure needs improvement

🎯 TPQC Decision Criteria:

Approve if: Countermeasure is complete, appropriate, and prevents recurrence

Reject if: Action insufficient, incomplete, or doesn't address root cause
